### 测试故事
- 点击父级菜单会toggle所有子菜单以及该父菜单的状态
- 点击单一子菜单时会toggle该子菜单的状态
- 当父级为选中状态时，点击其子菜单取消其选中，父级也会取消选中
- 点击子菜单至某一个父级菜单的所有子菜单被选中时，其父级也会被选中
- 点击右侧的清除按钮会清除掉所有选中状态
- 菜单上的文字较多时会省略号显示
- 当左侧菜单较多时，可以下滑查看屏幕中显示不到的菜单
